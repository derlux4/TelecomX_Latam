# TelecomX - Análisis de Evasión de Clientes

## Descripción
Este proyecto tiene como objetivo analizar los factores que influyen en la tasa de cancelación (churn) de clientes en la empresa Telecom X, utilizando técnicas de ciencia de datos con Python.

## Estructura del proyecto
- `/data`: Datasets del proyecto.
- `/notebooks`: Archivos de análisis (Google Colab).
- `/visualizations`: Gráficos generados.

## Objetivos
- Identificar patrones en los clientes que abandonan la empresa.
- Proporcionar insights para reducir la tasa de evasión.
